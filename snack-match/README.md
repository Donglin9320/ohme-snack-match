# OHME! Snack Match

Consumer-facing OHME! product discovery website.

The page combines OHME brand storytelling, five Qualtrics-derived purchase questions, exact pack formats, current public prices, and source-linked category benchmarks.

## What It Shows

- Customer Drivers: pre-survey hypothesis dashboard for why shoppers choose healthy snacks.
- Snack Match: five purchase questions that recommend available OHME products.
- Pack clarity: exact sizes, resealability, and current public prices.
- OHME moments: on-the-go, topping, lunchbox, and sharing entry points.
- Benchmarks: source-linked comparisons separated by role rather than ranked as direct equivalents.

## How to Use It

Start with an OHME moment or go directly to Snack Match, answer all five questions, then use the result links to explore the live OHME product listing.

## Files

- `index.html`: complete static site.
- `.nojekyll`: keeps GitHub Pages from applying Jekyll processing.
